<objective>
## Goal
Create `.base/data/` and `.base/hooks/` directories, move psmm.json and staging.json into `.base/data/`, and update all file path references so every tool, hook, and rule points to the new location.

## Purpose
Establishes the data directory foundation that all future data surfaces (active.json, backlog.json, user-created surfaces) will live in. Without this, Phase 2-5 have nowhere to put their files.

## Output
- `.base/data/` directory containing psmm.json and staging.json
- `.base/hooks/` directory (empty, ready for Phase 2)
- All CARL MCP tools reading/writing from `.base/data/`
- PSMM injector hook reading from `.base/data/`
- CARL rule 12 fallback path updated
- Zero functional regressions — all existing tools work identically
</objective>
